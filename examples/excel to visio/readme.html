Card Wall Generator

What is it?

this tool will take an excel spreadsheet with a hierarchical story list in it, and generate a "card wall" in visio from it.


a "card wall"

the spreadsheet might look like - business processes.xls

and the card wall might look like - TPS.vsd

Installation

You will need:

  • Windows - sorry only works on windows right now...
  • Excel
  • Visio
  • Ruby - download it from here - http://rubyforge.org/projects/rubyinstaller/

Once you have all those installed, you can test your installation by double clicking the "generate card walls.rb" file. It should open excel for a second, then open up visio and generate 2 card walls.

The Driver File

What the card wall generator does is configured by the "generate card walls.rb" file. Open it up and take a look. You should see stuff like : 

     data = load_excel_data 'Business Processes.xls', 1
     create_visio_document data, 'Caution.vsd'

You can customize this for your needs. Basically copy these 2 lines for every card wall you want to create, and substitute these for the values :

  • excel file with the master story list - "Business Processes.xls"
  • index of the sheet within the excel file - "1"
  • visio file that will be created - "Caution.vsd"
